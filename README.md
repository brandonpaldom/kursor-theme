# Kursor Theme

[![Visual Studio Marketplace](https://img.shields.io/visual-studio-marketplace/v/brandonpalmeros.kursor-theme)](https://marketplace.visualstudio.com/items?itemName=brandonpalmeros.kursor-theme)
[![GitHub](https://img.shields.io/github/stars/brandonpaldom/kursor-theme)](https://github.com/brandonpaldom/kursor-theme)

A collection of Cursor-inspired color themes for Visual Studio Code. These themes bring the clean, minimal aesthetic of Cursor to your VS Code experience.

![Kursor Theme Screen](https://res.cloudinary.com/dlomynswh/image/upload/v1768013997/kursor-theme/Captura_de_pantalla_2026-01-09_a_la_s_8.57.40_p.m..png)

## Themes

### Kursor Dark
A dark theme inspired by Cursor's default dark theme. Features a clean background with carefully chosen colors for syntax highlighting, providing excellent readability and a modern look.

### Kursor Light
A light theme inspired by Cursor's light theme. Perfect for bright environments or users who prefer light backgrounds. Maintains the same high contrast and readability as its dark counterpart.

### Kursor Dark High Contrast
A high contrast variant of the dark theme, designed for better accessibility. Uses stronger color differences to improve visibility for users with visual impairments.

### Kursor Dark Midnight
A midnight variant with deeper blues and Nord-inspired colors. Offers a cooler, more subdued palette while maintaining the Cursor aesthetic.

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Kursor Theme"
4. Click Install

### Manual Installation
1. Download the `.vsix` file from the [releases page](https://github.com/brandonpaldom/kursor-theme/releases)
2. In VS Code, go to Extensions (Ctrl+Shift+X)
3. Click the "..." menu and select "Install from VSIX..."
4. Select the downloaded `.vsix` file

## Usage
After installation, you can switch to any of the Kursor themes:
1. Open Command Palette (Ctrl+Shift+P)
2. Type "Color Theme"
3. Select "Preferences: Color Theme"
4. Choose your preferred Kursor theme
